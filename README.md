 ## Login & Registration System in C++

## Description
This project is a simple Login and Registration System developed in C++.  
It allows users to register with a username and password, and then log in using the same credentials.

User data is stored in a text file using file handling.

---

## Features
- User Registration
- Duplicate Username Check
- User Login Authentication
- File Handling (users.txt)
- Hidden Password Input (*)
- Error Handling for invalid login

---

## Technologies Used
- C++
- File Handling (fstream)
- Dev C++

---

## How to Run
1. Open the project in Dev C++
2. Compile the code
3. Run the program
4. Choose:
   - 1 for Registration
   - 2 for Login

---

## File Details
- main.cpp → Main program file
- users.txt → Stores usernames and passwords
